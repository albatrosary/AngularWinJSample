# AngularWinJSample

## Install

### Preparation

```
$ npm install -g http-server
```

### Setup

```
$ git clone git@github.com:albatrosary/AngularWinJSample.git
$ cd AngularWinJSample
$ npm install
```

### Demo

```
$ npm start
```

Launch the browser(http://127.0.0.1:9000/)
